# HR Recruitment Dashboard – Power BI Project

## Overview

The HR Recruitment Dashboard is a Power BI project designed to analyze and monitor the recruitment process of an organization. The dashboard helps HR teams track candidate applications, hiring sources, role-wise recruitment performance, and overall hiring insights through interactive visualizations.

This project demonstrates data visualization, HR analytics, dashboard storytelling, and business intelligence reporting using Power BI.

---

# Project Objective

The main goal of this dashboard is to:

* Monitor recruitment activities in real time
* Track candidate application progress
* Identify the most effective hiring sources
* Analyze recruitment trends role-wise
* Improve hiring decision-making using data insights

---

# Tools & Technologies Used

* Power BI Desktop
* Data Modeling
* Power Query
* DAX (Data Analysis Expressions)
* Interactive Visualizations

---

# Step-by-Step Project Workflow

## Step 1: Data Collection

Collected HR recruitment data from multiple sources such as:

* Candidate applications
* Hiring platforms
* Recruitment records
* Department hiring details

---

## Step 2: Data Cleaning

Used Power Query Editor to:

* Remove duplicates
* Handle missing values
* Standardize column formats
* Clean inconsistent records

---

## Step 3: Data Transformation

Performed transformations such as:

* Creating calculated columns
* Merging datasets
* Formatting dates
* Building relationships between tables

---

## Step 4: Data Modeling

Created relationships between tables to improve dashboard performance and reporting accuracy.

### Example:

* Candidate Table ↔ Recruitment Source Table
* Candidate Table ↔ Job Role Table

---

## Step 5: DAX Calculations

Created DAX measures for:

* Total Candidates
* Selection Rate
* Rejection Rate
* Source-wise Hiring Count
* Role-wise Recruitment Metrics

---

## Step 6: Dashboard Design

Designed an interactive dashboard with:

* Navigation buttons
* Filters and slicers
* KPI cards
* Charts and graphs
* User-friendly layout

---

## Step 7: Insight Generation

Analyzed the data to identify:

* Best hiring sources
* High-demand job roles
* Recruitment success trends
* Candidate conversion patterns
